# Steps

2. Run project creation skill
3. Set up version control
4. Create prototype GDD
5. Create prototype GDD roadmap
6. Create claude.md file


## Skills
1. godot-project-setup
2. version-control-helper
3. prototype-gdd-generator


## Commands
### On Project Setup
Set up Godot project structure using godot-project-setup skill.  
Create a prototype GDD using the prototype-gdd-generator skill
Create roadmap for prototype using prototype-roadmap-planner skill
Set up my local git repo using version-control-helper skill

### For Each Feature
Help me design this feature using the feature-idea-designer skill
Generate a feature spec using feature-spec-generator skill for the feature 
Implement the feature spec using the feature-implementer skill for the feature
Update architecture doc using architecture-documenter skill
Commit to local git repo and github repo using version-control-helper skill

### Debugging
Debug this using error-debugger skill 

### Refactoring
Separate data from logic using the data-driven-refactor skill
Implement the data extraction recommendations using data-extractor skill

### Asset Pipeline
Create a character using pixellab-create-character skill
